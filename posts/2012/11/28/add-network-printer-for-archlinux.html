终于，实验室买了台高级打印机，可以配置独立的IP地址，支持网络打印。 索性给我这台Arch Linux也配置上网络打印功能。

UNIX/Linux下的打印管理工具CUPS

CUPS(Common UNIX Printing System)，是Apple开发的一套UNIX开源打印系统。 Arch下可可以通过CUPS进行打印管理。

安装CUPS很简单：

<<<<<<< HEAD 
pacman -S cups libcups
=======

pacman -S cups libcups
>>>>>>> gh-pages





CPUS安装好后，需要根据打印机安装相应的driver。
我们的HP打印机对应的驱动包hplip，所有的HP打印机都可以使用：

<<<<<<< HEAD

pacman -S hplip
=======

pacman -S hplip
>>>>>>> gh-pages





之后启动CUPS守护进程，并把CUPS加入启动服务。

<<<<<<< HEAD

systemctl start cups
systemctl enable cups
=======

systemctl start cups
systemctl enable cups
>>>>>>> gh-pages





CUPS的基本管理功能通过本地web页面进行操作，需要前用户在sys组，在这里临时把当前用户加入sys：

<<<<<<< HEAD

usermod -aG sys [USERNAME]
=======

usermod -aG sys [USERNAME]
>>>>>>> gh-pages





打开浏览器http://localhost:631，就可以看到CUPS的配置页面了。


UI做的比较简洁，直接到Administration页面Add Printer即可。
配置好后打印时就能看到刚刚添加的打印机了。
